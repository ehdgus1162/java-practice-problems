## 📚 문제 2: 메소드 호출 흐름 및 재귀 호출

### ✍️ **문제 설명**
- `main` 메소드에서 재귀 메소드를 호출하여 1부터 `n`까지의 합을 출력하는 프로그램을 작성하세요.
- 재귀 호출 과정에서 호출 흐름을 출력하도록 하세요.

---

### 🎯 **목표**
- 재귀 메소드를 사용하여 1부터 `n`까지의 합을 계산합니다.
- 각 재귀 호출 시, 현재 `n` 값을 출력하여 호출 흐름을 확인할 수 있도록 합니다.

---

### ⚙️ **요구사항**
1. 재귀 메소드의 정의와 호출 흐름을 출력할 것.
2. `n`이 1이 될 때까지 재귀 호출을 반복할 것.
3. 최종 결과로 1부터 `n`까지의 합을 출력할 것.

---

### 🔗 **제약 조건**
- `n`은 1 이상의 정수입니다.
- 반복문(`for`, `while`) 사용 없이 재귀를 활용하여 구현합니다.
