# 📘 문제 1: 단독 if문과 중첩 if문 활용

## 📝 문제 설명
사용자로부터 정수를 입력받아 다음과 같은 조건에 따라 출력하는 프로그램을 작성하세요.

### ✅ 요구사항
1. 입력된 값이 **양수인지 음수인지** 확인합니다.
2. **양수인 경우**, **짝수인지 홀수인지**를 추가로 출력합니다.

- 하나의 정수 (예: `8`)
