## 🔧 **문제 2: 산술 복합 대입 연산자와 복합 연산 활용**

### 📝 **문제 설명**
- **정수형 변수**를 하나 선언하고, 다음 연산을 순차적으로 수행한 결과를 출력하세요.
    1. **10을 더함**
    2. **3으로 나눔**
    3. **5를 곱함**
    4. **8을 뺌**
    5. **2로 나눈 나머지를 구함**

### 📥 **입력 예시**
초기값: 20


### 📤 **출력 예시**
+= 10 -> 30 /= 3 -> 10 *= 5 -> 50 -= 8 -> 42 %= 2 -> 0


### 💡 **힌트**
- **복합 대입 연산자(`+=`, `/=`, `*=`, `-=`, `%=`)**를 사용하세요.
- 각 단계별 출력 시 **설명과 결과를 함께 출력**하세요.
