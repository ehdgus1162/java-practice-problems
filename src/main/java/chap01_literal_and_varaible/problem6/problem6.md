### 🚨 **문제 6: 오버플로우 및 언더플로우 탐색**

#### ✍ **문제 설명**
- `int`형 변수의 최대값과 최소값을 출력하고, 오버플로우 및 언더플로우가 발생하는지 확인하세요.

#### 🖨 **출력 예시**
int의 최대값: 2147483647 int의 최소값: -2147483648 int 최대값 + 1: -2147483648 (오버플로우 발생) int 최소값 - 1: 2147483647 (언더플로우 발생)



#### 💭 **힌트**
- `Integer.MAX_VALUE`와 `Integer.MIN_VALUE`를 활용하세요.
- 오버플로우와 언더플로우의 개념을 이해하고, 실제 연산을 통해 결과를 확인하세요.
