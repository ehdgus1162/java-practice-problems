## 🚀 **문제 6: 논리 연산 우선순위 실습**

### 📝 **문제 설명**
- 다음 **복합 논리식**을 계산하고 결과를 출력하세요.
    1. `(7 > 5 && 3 < 2) || !(4 == 4 && 9 > 6)`
    2. `!(true || false) && (10 >= 10 && 5 != 3)`

### 📤 **출력 예시**
결과 1: false 결과 2: false


### 💡 **힌트**
- **괄호(`()`)**를 사용하여 연산자 우선순위를 명확하게 표현하세요.
- **부정 연산자(`!`)**의 적용 순서를 확인하세요.
