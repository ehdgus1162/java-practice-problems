### 🌟 **[심화 문제: Java 객체 지향 프로그래밍 종합 문제]** 🌟

---

## 📝 **문제 설명**
다음 요구사항을 충족하는 **종합적인 Java 프로그램**을 작성하세요.  
**클래스 설계, 캡슐화, 생성자, 메소드 오버로딩, static 및 final 키워드, 초기화 블록**과 같은 개념을 **적극적으로 활용**해야 합니다.

---

## 🎯 **문제 요구사항**

### ✅ **1단계: 클래스 설계 및 인스턴스 생성**
1. **Vehicle (탈것)**라는 **추상 클래스**를 생성하세요.
    - **private 필드**: `brand (브랜드)`, `model (모델)`, `year (연식)`
    - **생성자**: 위 필드를 초기화하는 생성자
    - **추상 메소드**: `start()`, `stop()`

2. **Car (자동차)**와 **Motorcycle (오토바이)** 클래스를 `Vehicle` 클래스를 **상속**하여 구현하세요.
    - 각각의 `start()`와 `stop()` 메소드를 **오버라이딩**하여, 차량 시작 및 종료 시 출력할 메시지를 작성하세요.
    - **Car 클래스**는 `maxSpeed`(최대 속도)와 `acceleration()` 메소드를 추가하여, 속도를 증가시키는 기능을 제공합니다.
    - **Motorcycle 클래스**는 `type`(예: 스포츠, 크루저 등) 필드를 추가하고, 정보를 출력하는 메소드를 작성하세요.

---

### ✅ **2단계: 캡슐화 및 안전한 데이터 관리**
1. **BankAccount (은행 계좌)** 클래스를 생성하세요.
    - **private 필드**: `balance (잔액)`
    - **생성자**: 초기 잔액을 설정하는 생성자
    - **메소드**:
        - `deposit(int amount)` - 잔액에 금액을 추가
        - `withdraw(int amount)` - 잔액에서 금액을 인출 (단, 잔액 부족 시 오류 메시지 출력)
    - ⚡ **음수 금액에 대한 예외 처리**를 반드시 포함하세요.

---

### ✅ **3단계: 정적(static) 키워드 및 상수(final) 활용**
1. **VehicleRegistry (차량 등록소)** 클래스를 생성하세요.
    - **static 필드**: `totalVehicles (총 등록된 차량 수)`
    - **static 메소드**: `registerVehicle(Vehicle vehicle)` - 차량을 등록할 때마다 `totalVehicles` 값을 증가시키고, 등록된 차량의 정보를 출력하세요.
    - **final 상수**: `MAX_VEHICLES = 100` - 등록할 수 있는 최대 차량 수를 제한하고, 초과 등록 시 경고 메시지를 출력하세요.

---

### ✅ **4단계: 초기화 블록 및 생성자 활용**
1. **Owner (소유자)** 클래스를 생성하세요.
    - **필드**: `name`, `age`, `BankAccount account`
    - **초기화 블록**: 소유자 정보 초기화 전에 **"소유자 정보를 초기화합니다."**라는 메시지를 출력합니다.
    - **생성자**: 이름과 나이를 초기화하는 생성자
    - **메소드**: `purchaseVehicle(Vehicle vehicle, int price)` -
        - 차량 구매 시 **계좌 잔액이 충분한지 확인**하고, 구매가 성공하면 차량 등록소에 등록하세요.
        - 부족한 경우 **"잔액이 부족합니다."** 메시지를 출력하세요.

---

### ✅ **5단계: 메소드 오버로딩 및 다형성 구현**
1. **Utility (유틸리티)** 클래스를 생성하고, 다음 메소드 오버로딩을 구현하세요:
    - `displayInfo(Car car)`
    - `displayInfo(Motorcycle motorcycle)`
    - `displayInfo(Vehicle vehicle)`

   각 메소드에서 **다형성을 활용하여 차량의 고유한 정보를 출력**합니다.

---

## 💡 **추가 심화 과제 (선택 과제)**
- 🚀 **초기화 블록, static 초기화 블록**을 사용하여 클래스가 처음 로드될 때 메시지를 출력하세요.
- 🔄 **VehicleRegistry**에 **배열 또는 컬렉션(List)**을 사용하여 등록된 차량 목록을 저장하고, 전체 등록 차량 정보를 출력하세요.
- 🔒 **BankAccount**에 **getter 메소드**만 제공하여 외부에서 직접 잔액 변경이 불가능하도록 하세요.
