## ✅ 문제 1: 메소드 오버로딩 이해 및 구현

### 💡 설명
- 두 수의 합을 반환하는 메소드를 작성하세요.
- 다음과 같이 메소드 오버로딩을 활용하여 메소드를 구현하세요:
    - 정수형 두 수의 합
    - 실수형 두 수의 합
    - 세 개의 정수 합

### 🔎 입력 예시
```java
add(5, 7);
add(5.5, 3.5);
add(5, 7, 3);
📝 출력 예시


두 수의 합(정수형): 12
두 수의 합(실수형): 9.0
세 수의 합: 15
💡 힌트
메소드 오버로딩은 메소드 이름이 같지만 매개변수의 타입이나 개수가 다른 경우를 말합니다.
